# Transport & Logistics Dashboard (Power BI)

An interactive Power BI dashboard providing end-to-end visibility into logistics operations.

## Features
- 📊 KPI tracking: Total shipments, on-time delivery %, avg delivery time, delay days, cancellation rate
- 🚚 Shipment breakdown by transport mode (Air, Rail, Ship, Truck)
- 📅 Monthly trend analysis: delay days vs. shipment volume
- 🎯 Priority segmentation: Express, Standard, Economy
- 🗺️ Origin–Destination matrix for shipment volume and average delay
- 🔍 Interactive slicers: Status, Mode, Origin, Destination

## Tools Used
- Power BI Desktop
- DAX for calculated measures
- Data modeling with transport/logistics dataset
