# 🏥 Hospital Dashboard

## 📌 Overview
This project provides an interactive **Hospital Dashboard** designed to analyze patient data and visualize key healthcare metrics. It helps hospital administrators, doctors, and analysts monitor patient outcomes, departmental performance, and treatment costs.

---

## ✨ Features
- **Key Metrics**
  - Total Patients: 200
  - Average Stay Duration: 8.07 days
  - Mortality Rate: 3.5%
  - Readmission Rate: 13%

- **Filters**
  - Outcome: Deceased, Readmitted, Recovered, Under Treatment
  - Gender: Male, Female
  - Age Groups: Child, Teen, Young Adult, Adult, Middle Age, Senior

- **Visualizations**
  - Patient distribution by department (Orthopedics, General Medicine, Pediatrics, Cardiology, Neurology, Oncology)
  - Monthly patient counts (Jan–Dec)
  - Average treatment cost by age group
  - Top 5 diagnoses (Arthritis, Diabetes, Epilepsy, Heart Disease, Hypertension)
  - Patient outcomes (Recovered, Under Treatment, Readmitted, Deceased)

---

## 📊 Dashboard Insights
- Majority of patients are **Recovered (63.5%)**.
- **Average treatment cost** ranges between 9K–11K depending on age group.
- **Orthopedics and General Medicine** departments handle the highest patient volume.
- **Readmission rate (13%)** highlights areas for improvement in post-treatment care.

---

## 🛠️ Tech Stack
- **Power BI / Excel** for dashboard creation
- **SQL/MySQL** for data extraction and transformation
